NHS RTT Performance Analysis Project

This project analyses NHS Referral to Treatment (RTT) waiting times using Python. The analysis includes national trends, provider-level performance, and specialty-level long-wait percentages. A dashboard summarises key insights.

Objectives
Analyse national RTT waiting list trends over time
Evaluate performance against the NHS 18-week treatment target
Identify providers with the highest proportion of long-wait patients
Identify specialties with the highest long-wait percentages
Create a visual dashboard summarising key insights

Tools & Technologies
Python
Pandas
Matplotlib
Jupyter Notebook


Files Included
NHS RTT Performance Analysis.ipynb → Main analysis notebook
RTTed.xlsx Data files used for analysis (CSV/XLSX)
NHS RTT Dashboard.pdf → Exported dashboard visualisation
NHS RTT Performance Readme.txt readme file of the project

How to Run the Project
Clone the repository
Open the notebook NHS_RTT_Analysis.ipynb
Install required libraries (pandas, matplotlib)
Run all cells sequentially
